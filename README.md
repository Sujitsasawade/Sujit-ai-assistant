# AI Chatbot using Llama 3 and Ollama

This project is a simple AI chatbot that allows users to ask questions and receive intelligent responses. The chatbot is built using HTML, CSS, and JavaScript and connects to a locally running AI model.

The AI model runs using Ollama and Llama 3, which generates responses to user questions in real time.

## Features

- Chat interface similar to ChatGPT
- Real-time AI responses
- Runs locally without paid APIs
- Simple and clean UI
- Automatic chat scrolling

## Technologies Used

- HTML
- CSS
- JavaScript
- Ollama
- Llama 3
